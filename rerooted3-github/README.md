# ReRooted 🌱

Attachment-aware relationship growth through emotional safety, structured intimacy, and repair that your nervous system can trust.

## ✨ Features

- **Attachment Quiz**: Discover your attachment style (anxious, avoidant, fearful-avoidant, or secure)
- **Premium Insights**: Deeply personalized breakdown of your attachment pattern
- **Daily Rituals**: Nervous-system safe practices tailored to your attachment type
- **Connection Reps**: Small, structured intimacy practices
- **Educational Content**: Learn about attachment patterns and relationship dynamics
- **Progressive Web App**: Install on your phone's home screen for app-like experience

## 🚀 Quick Deploy

### FASTEST: Netlify Drop (60 seconds)
1. Visit **https://app.netlify.com/drop**
2. Drag and drop the `out` folder
3. Get instant URL: `https://[random-name].netlify.app`

### Deploy to Netlify via GitHub
1. Push this repo to GitHub
2. Visit **https://app.netlify.com**
3. Click "Import from Git"
4. Select your repo
5. Deploy automatically

### Deploy to Vercel
1. Push this repo to GitHub
2. Visit **https://vercel.com**
3. Click "Import Project"
4. Select your repo
5. Deploy automatically

**See `LIVE_URL_INSTRUCTIONS.md` for detailed step-by-step guides!**

## 📱 Mobile-First PWA

This app is fully optimized for mobile devices and can be installed as a Progressive Web App:

### iPhone Installation:
1. Open the deployed URL in Safari
2. Tap the Share button
3. Tap "Add to Home Screen"
4. Your app will open in standalone mode (no browser UI)

### Android Installation:
1. Open the deployed URL in Chrome
2. Tap the three dots menu
3. Tap "Add to Home Screen" or "Install App"
4. Your app will open in standalone mode

### PWA Features:
- ✅ Works offline (after first visit)
- ✅ Full-screen standalone mode
- ✅ Native app-like experience
- ✅ Fast loading and smooth animations
- ✅ Mobile-optimized touch interactions

## 🛠️ Local Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm run start
```

Visit `http://localhost:3000` to see your app.

## 🎨 Tech Stack

- **Framework**: Next.js 13 with App Router
- **Styling**: Tailwind CSS
- **UI Components**: shadcn/ui + Radix UI
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **TypeScript**: Full type safety
